# stack

## 介绍
stack 类是一种容器适配器，它给予程序员栈的功能——特别是 FILO（先进后出）数据结构。

该类模板表现为底层容器的包装器——只提供特定函数集合。栈从被称作栈顶的容器尾部推弹元素。
## 任务
实现stack 最基本的功能，例如push，pop，empty，size，swap

如果你对c++了解更多，例如知道运算符重载，那就试试实现按照字典顺序比较两个 stack 的值吧。
## 函数要求
| **函数**        | **功能**       |
|-----------------|--------------|
| push | 向栈顶插入元素 |
| top |	访问栈顶元素 |
| empty | 检查容器是否为空 |
| size | 返回元素数 |
| pop | 移除栈顶元素 |
| swap | 交换内容 |
